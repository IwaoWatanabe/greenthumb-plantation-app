# Greenthumb Nursery 管理システム（日本語）

本リポジトリは、Java Swing と MySQL によるデスクトップ型植物園（ナーサリー）管理アプリケーションです。管理者、スタッフ、顧客の各役割に対応し、在庫・注文管理・レポート機能を備えています。

クイックスタート
1. MySQL を起動し、SQL をインポート:
   mysql -u root -p greenthumb_nursery < greenthumb_nursery.sql
2. JAR を実行:
   java -jar greenthumb-nursery-1.0.0.jar
3. デフォルトアカウント:
   - admin / admin123
   - staff1 / staff123
   - customer1 / customer123

詳細な手順は README.md（英語）を参照してください。
